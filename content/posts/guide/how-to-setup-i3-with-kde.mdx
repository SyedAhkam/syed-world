## Heading

### Install i3

```bash
sudo pacman -S i3-gaps i3blocks i3lock i3status
```

### Install KDE

```bash
sudo pacman -S plasma kde-applications
```

### Configure i3

#### Configure i3 to use KDE's settings

```bash
echo "export KDEWM=/usr/bin/i3" >> ~/.xinitrc
```
